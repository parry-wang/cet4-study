## 1. 产品概述

英语四级智学平台是一款面向备考大学英语四级（CET-4）学生的综合性学习网站，提供单词背诵、听力播报、阅读理解、作文书写、试卷生成五大核心模块，帮助学生系统化备考、查漏补缺。
- 解决传统四级备考资料分散、缺乏系统化训练与错题积累的问题，提供一站式学习平台
- 面向在校大学生及自考人群，目标是提升四级考试通过率，建立科学的学习闭环

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 普通用户 | 本地免登录（首次访问自动建档） | 使用全部学习模块、查看个人学习进度与错题本 |
| 访客 | 无需注册 | 浏览首页与功能介绍 |

### 2.2 功能模块
1. **首页（学习仪表盘）**：今日学习任务、学习进度概览、快速入口、学习日历热力图
2. **单词背诵**：每日50词打卡、词卡翻转、错词自动入库、间隔重复复习、词汇进度统计
3. **听力播报**：分级听力素材、播放控制、原文对照、听力测验
4. **阅读理解**：分级阅读篇章、题目作答、解析反馈、计时训练
5. **作文书写**：话题题库、写作编辑器、字数统计、范文对照、写作模板
6. **试卷生成**：自定义题型与题量、自动组卷、在线作答、批改评分
7. **错题本**：单词/阅读/听力错题汇总、复习再测、掌握度标记

## 3. 核心流程

**主要用户流程**：
1. 用户进入首页，查看今日50词任务及各模块进度
2. 进入单词背诵，学习当日词库（翻转词卡→考察模式→错题自动入库）
3. 错词进入复习队列，按间隔重复算法在后续日期反复出现
4. 用户根据计划依次完成听力、阅读、写作训练
5. 阶段性使用试卷生成模块进行模拟测试，错题汇总至错题本
6. 通过错题本复习再测，巩固薄弱环节

## 4. 用户界面设计

### 4.1 设计风格
- **整体风格**：编辑杂志风（Editorial Magazine）—— 学院典雅与现代克制的结合，致敬经典英文书籍排版
- **主色调**：墨纸奶白（#F7F3EC 背景 / #1A1A1A 墨色正文），点缀深酒红（#7A2E2E）作为强调色，传达学术庄重感
- **辅助色**：旧金（#B8860B）用于进度与成就，苔绿（#3E5641）用于"已掌握"状态
- **按钮风格**：方角微圆（2px），细描边按钮为主，主操作按钮为深酒红实心
- **字体**：标题用 Fraunces（衬线展示字体，文学气质）；正文用 Source Serif 4（衬线，阅读舒适）；UI 辅助用 JetBrains Mono（等宽，用于数据/计时）
- **布局**：桌面优先，顶部固定导航 + 双栏/三栏卡片网格，大量留白与细分隔线
- **图标/emoji**：使用细线 SVG 图标（Lucide 风格），克制不滥用

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |

| 试卷生成 | 组卷配置 | 表单式配置面板，实时预览缩略 |
| 错题本 | 错题列表 | 表格化布局，三色掌握度标签 |

### 4.3 响应式
- 桌面优先（≥1280px 完整三栏体验）
- 平板（768-1280px）降级为双栏
- 移动端（<768px）单栏堆叠，导航折叠为汉堡菜单，触控优化按钮尺寸

### 4.4 3D 场景指导
- 不适用（本项目为 2D 学习平台，无 3D 场景需求）
